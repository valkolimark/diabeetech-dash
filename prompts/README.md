# Diabeetech (B-Tech) — Session Prompts

Store new session prompts here for future Claude Code sessions.

## How to Use

1. Create a new prompt file: `session-YYYY-MM-DD-description.md`
2. Paste the prompt content at the start of a new Claude Code session
3. The prompt should reference `docs/APP-OVERVIEW.md` for full context
